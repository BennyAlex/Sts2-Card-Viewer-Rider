# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## 1.0.0
- Card grid browser with portraits, cost, type, and rarity
- Relic list view with icons and descriptions
- Filter by name, card type, and rarity
- Language switching (auto-discovers localization folders)
- Editor card preview via gutter icon when opening card .cs files
- Auto-detect STS2 mod projects via Sts2PathDiscovery.props
